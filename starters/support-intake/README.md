# Support Intake Starter

ECZ-ID-backed Support Intake agent scaffold.

## Prerequisites
- ECZ-ID Business Passport + Agent Credential (from TrustOps)
- See full requirements at [developers.ecocitizenz.com/agent-starter-kits/support-intake](https://developers.ecocitizenz.com/agent-starter-kits/support-intake)

## Quick Start
1. Copy this directory
2. `cp .env.example .env`
3. Go to [TrustOps](https://trustops.ecocitizenz.com/start) to acquire credentials
4. Fill in `.env` with your TrustOps-issued credentials
5. Follow the build guide on the Developer Gateway

## Required Passports
- ECZ-ID Business Passport (parent)
- ECZ-ID Agent Credential
- Additional domain-specific passports as specified in the build guide

## Important
Credentials are acquired through TrustOps, not GitHub. Verification is through Resolver.
