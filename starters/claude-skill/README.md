# Claude Skill Starter

Build an ECZ-ID-backed Claude skill (tool use pattern) with verified identity.

## Prerequisites
- Anthropic API access
- ECZ-ID Business Passport + Agent Credential (from TrustOps)

## Quick Start
1. Copy this directory
2. `cp .env.example .env`
3. Go to [TrustOps](https://trustops.ecocitizenz.com/start) to acquire credentials
4. Fill in `.env` with your TrustOps-issued credentials
5. Follow the build guide at [developers.ecocitizenz.com/agent-starter-kits/claude-skill](https://developers.ecocitizenz.com/agent-starter-kits/claude-skill)

## Required Passports
- ECZ-ID Business Passport (parent)
- ECZ-ID Agent Credential

## Important
Credentials are acquired through TrustOps. Verification is through Resolver.
